# Project Docs

## Arquivos principais

- `handoff.md`
  - resumo do estado atual do projeto
  - decisões já tomadas
  - próximo passo esperado

- `pixel-art-assets.md`
  - checklist completo de assets para repaginar o app em pixel art
  - referência para UI, ícones, personagem, inventário, efeitos e navegação

## Como usar no outro computador

Peça ao Codex para:

1. clonar o repositório
2. ler `docs/handoff.md`
3. ler `docs/pixel-art-assets.md`
4. continuar o projeto a partir desse contexto
